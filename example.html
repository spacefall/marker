Marker


this is a really basic example of marker and how it can be used to create a simple document

Usage


Marker takes heavy inspiration from markdown, but tries to do everything slightly differently because why not? (also because i think it's nicer to write, but that's probably just me)

Inline "Tags"


Inline tags are used to format the text in a line. (crazy concept, right?)

The following tags are available:
bold: **bold**
italic: //italic//
underline: __underline__
strikethrough: --strikethrough--
inline code: ^^inline code^^
superscript: ''superscript''
subscript: ,,subscript,,

Header "Tags"


Headers are slightly different from markdown, they start with a # but then continue with a number and a space. This number is the level of the header and it can be from 1 to 6.

Additional features


The "\" character can be used to escape any of the formatting tags, even a new line, so you can split your text in multiple lines without affecting the output.
